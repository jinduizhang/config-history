### 2. 代码规范优化

#### 2.1 变量命名规范化

**核心原则**: 禁止使用数字后缀命名变量，使用语义化英文标识

| 场景 | 旧命名 | 新命名 |
|------|--------|--------|
| 版本对比-源 | `version1`, `v1` | `sourceVersion`, `sourceHistory` |
| 版本对比-目标 | `version2`, `v2` | `targetVersion`, `targetHistory` |
| 快照数据 | `snapshot1/2` | `sourceSnapshot/targetSnapshot` |
| 配置值 | `value1/2` | `sourceValue/targetValue` |
| JSON 字符串 | `json1/2` | `sourceJson/targetJson` |
| Map 对象 | `map1/2` | `sourceMap/targetMap` |
| 对象比较 | `obj1/2` | `first/second` |

#### 2.2 涉及文件

**后端:**
- `DiffResponse.java` - sourceVersion/targetVersion, sourceValue/targetValue
- `DiffResult.java` - sourceVersion/targetVersion, sourceSnapshot/targetSnapshot
- `ConfigServiceImpl.java` - sourceHistory/targetHistory, sourceJson/targetJson
- `HistoryService.java` - sourceHistory/targetHistory
- `DiffCalculator.java` - sourceSnapshot/targetSnapshot
- `DiffCalculatorImpl.java` - sourceSnapshot/targetSnapshot, sourceMap/targetMap
- `HistoryRecorderImpl.java` - first/second

**前端:**
- `types/index.ts` - sourceVersion/targetVersion, sourceValue/targetValue
- `ShellHistoryPanel.vue` - sourceContent/targetContent
- `ShellHistoryModal.vue` - sourceContent/targetContent
- `HistoryList.vue` - sourceVersion/targetVersion

### 3. Bug 修复

#### 3.1 回退操作保存错误值

**问题**: 回退操作时，新版本记录保存的是回退前的值，而不是回退后的目标值

**修复**: `ConfigServiceImpl.java` 的 `rollback()` 方法
```java
// 修复前（错误）
rollbackHistory.setConfigValue(current.getConfigValue());  // 保存回退前的值
current.setConfigValue(targetVersion.getConfigValue());

// 修复后（正确）
current.setConfigValue(targetVersion.getConfigValue());    // 先更新配置
configItemMapper.updateById(current);
rollbackHistory.setConfigValue(targetVersion.getConfigValue());  // 再记录历史
```

### 测试数据创建

```bash
# 创建配置
curl -X POST http://localhost:8080/api/v1/configs \
  -H "Content-Type: application/json" \
  -d '{"configKey":"test","configName":"Test","configValue":"v1","operator":"admin"}'

# 更新配置（多次）
curl -X PUT http://localhost:8080/api/v1/configs/2 \
  -H "Content-Type: application/json" \
  -d '{"configKey":"test","configName":"Test","configValue":"v2","operator":"admin"}'

# 查看历史
curl http://localhost:8080/api/v1/configs/2/history

# 版本对比
curl "http://localhost:8080/api/v1/configs/2/diff?from=2&to=3"

# 回退
curl -X POST "http://localhost:8080/api/v1/configs/2/rollback/2?operator=admin"
```
